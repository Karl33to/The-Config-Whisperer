# Examples

This directory contains sample files to help you get started with **The Config Whisperer**.

## 📂 Contents

### Rulesets (`rulesets/`)
JSON files containing validation rules.
- **`bf-kpv.json`**: Personal ruleset for Open Racers.
- **`bf-spec-racing.json`**: GOTF 2025 Full Prescriptive Ruleset.
- **`blank.json`**: A starter template that ignores comments.

### Configs (`configs/`)
Text files containing configuration exports.
- **`bf-kpv.txt`**: Sample config matching the KPV ruleset.
- **`bf-spec-racing.txt`**: Sample config for the GOTF 2025 spec.
- **`betaflight_sample.txt`**: A generic Betaflight 4.5 dump snippet.

## 🚀 How to use them
1. Open the [Live Tool](https://Karl33to.github.io/The-Config-Whisperer/).
2. Drag a file from `rulesets/` into the **Ruleset (JSON)** panel.
3. Drag a matching file from `configs/` into the **Config Input** panel.
4. Click **Validate**.
